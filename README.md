# Stocked

Stocked is a smart household inventory app that helps users track supplies, manage shopping lists, and reduce waste.

It started as a simple inventory tracker and grew into a full app with shared households, recipes, analytics, and smart features.

## Features

- Inventory management
  - Add, edit, delete items
  - Categories and custom units
  - Low stock warnings
  - Expiry tracking

- Smart shopping list
  - Automatically adds low stock items
  - Updates when items are restocked
  - Live updates with Supabase

- Activity & notifications
  - Tracks changes
  - Alerts for low stock and expiring items

- Barcode scanning
  - Scan products with your camera
  - Saves barcode history

- Receipt scanner
  - Uses OCR to detect products from receipts

- Recipes
  - Checks ingredients against inventory
  - Shows missing items

- Analytics
  - Tracks consumption
  - Shows inventory usage

- Household mode
  - Create or join households
  - Share inventory with others
  - Demo mode for testing

- PWA
  - Installable app
  - Offline support

## Built With

- HTML
- CSS
- JavaScript
- Supabase
- OpenFoodFacts API
- Tesseract OCR

## Project

Stocked was built to solve a simple problem: keeping track of what you have at home.

What started as a dashboard with a few inventory cards became a complete household management app with shopping, recipes, analytics, and sharing.
